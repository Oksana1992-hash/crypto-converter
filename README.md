# Конвертер криптовалют (Crypto Exchange Rate App)

Простое и удобное десктопное приложение на Python с графическим интерфейсом (GUI) для быстрого получения актуальных курсов популярных криптовалют по отношению к фиатным деньгам (USD, EUR, RUB). 

Данные о стоимости валют запрашиваются в реальном времени через официальное публичное API сервиса **CoinGecko**.

---

## ✨ Функционал приложения

* 🔄 **Обновление курсов в реальном времени**: данные всегда актуальны на секунду нажатия кнопки.
* 🧮 **Подсчет произвольного количества**: вы можете ввести любое количество криптомонет (например, `0.23` или `5`), приложение автоматически пересчитает итоговую стоимость.
* 🇷🇺 **Русскоязычный интерфейс**: при выборе кода валюты приложение интерактивно подсказывает её полное название на русском языке.
* 🛡️ **Защита от ошибок ввода**: списки выбора заблокированы от ввода случайного текста вручную, а поле количества проверяет корректность введенных чисел.
* ⏱️ **Таймаут сетевых запросов**: если интернет-соединение пропадет, приложение не зависнет, а выдаст аккуратное предупреждение.

## 🛠️ Требования к окружению

Для запуска приложения вам понадобятся:
* **Python** версии 3.10 или выше.
* Модуль **Tkinter** (обычно поставляется в комплекте с Python по умолчанию).
* Библиотека **requests** для отправки сетевых запросов.

## 📦 Инструкция по установке и запуску

### Шаг 1. Склонируйте репозиторий и перейдите в корень проекта.

```bash
git clone https://github.com/Oksana1992-hash/crypto-converter.git
cd путь_к_вашей_папке
```

### Шаг 2. Установите необходимые зависимости.
Установите библиотеку `requests` через диспетчер пакетов `pip`:
```bash
pip install requests
```

### Шаг 3. Запустите приложение

```bash
python3 crypto-converter.py
```
